# LuxLink-QE-Simulator
LuxLink simulates quantum entanglement using an Arduino Uno. Two LEDs flicker in independent superposition until a button press triggers a "measurement," forcing the wavefunction to collapse. This hardware-logic bridge demonstrates Bell-state correlation, in which separate bits snap into identical, synchronised realities.
